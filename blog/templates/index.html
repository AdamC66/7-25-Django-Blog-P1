what is a client?

A client is a computer that obtains information and applications from a server

what is a server?

a server is a computer/program that manages access to a resource and provides responses to client requests

what is an HTTP request?

an HTTP request is a request from client to server over the hypertext transfer protocol

what is an HTTP response?
